# Hardhire — Frontend

Next.js landing page and web application.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|---|---|
| `/` | Landing page with search, value prop, and CTAs |
| `/homeowners` | Homeowner-focused features page |
| `/contractors` | Contractor directory |
| `/contractor/[id]` | Individual contractor report |
| `/search` | Search results |
| `/dashboard/homeowner` | Homeowner dashboard |
| `/dashboard/gc` | General contractor dashboard |
| `/signin` | Sign in |
| `/signup` | Sign up |

## Stack

- Next.js 13 (App Router)
- React 18
- Tailwind CSS
- Framer Motion (animations)
- Recharts (charts)
- Radix UI (accessible primitives)
- Lucide React (icons)
